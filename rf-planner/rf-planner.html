Agentic RF Planner API (1.0.0)

Download OpenAPI specification:

REST API for the Agentic RF Planner web application — planning jobs, auth, mesh profiles, and RF parameters.

Protected routes require Authorization: Bearer <token> (Cognito JWT stored in localStorage).

Planning

Core RF coverage planning. The UI calls POST /api/plan/submit then polls GET /api/plan/status/{job_id} for each TX in the queue. Results drive heatmap, sector overlay, and PolyEdge marker rendering.

Run RF Planning

Runs the full RF planning pipeline for a transmitter location. Returns a coverage grid, heatmap data, sector overlays, and optional UE placement recommendations.

Set ray_mode to 2d for fast 2D propagation or 3d / 3d_rt for height-aware multipath ray tracing. Pass sectors for non-omnidirectional coverage (angle-based or polygon).
