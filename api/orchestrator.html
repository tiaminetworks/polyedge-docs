PolyEdge Radio Configuration & Orchestration API (1.1)

Download OpenAPI specification:

REST API used by the PolyEdge web GUI (GUI/server.js) to configure the 5G NR UE radio, passive radar pipeline, and RF processing, and to start/stop/monitor the managed processes (init_nrUE, stream_main). See RADIO_CONFIGURATION_API.md for the full narrative documentation, operational workflows, and Socket.IO events (not covered by this REST-only spec).
