PolyEdge Detection API (27.0.0)

Download OpenAPI specification:

The Detection API provides poll-based access to PolyEdge's passive radar detections: target positions, radar returns, ML predictions, and drone/RID detections. It serves the same underlying data as the WebSocket stream, for consumers that prefer request/response polling over a persistent connection.

Detections

Get the full detection message

The complete detection message for the most recent CPI: radar returns, ML predictions, RID (drone) detections, fused coordinates, and metadata, in the same shape as the WebSocket stream. Use this when you want everything in one call; use the narrower endpoints below when you only need one part of it.
