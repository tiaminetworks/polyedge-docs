Targets

Live PolyEdge detections polled by the dashboard every 200 ms. Drone data includes ISAC, RID, and NTN targets. Person data includes ground tracks.

Get Drone Detections

Returns the latest PolyEdge drone detections for the authenticated user's device. Data is read from the per-tenant DynamoDB table referenced in UserLoginData.

Supports multiple detection modes:

  • Mode 1 (ISAC): passive radar localization
  • Mode 2 (RID): Remote ID detections
  • Mode 3: NTN / non-terrestrial targets (target_flag = 3)

Mode 3 entries with trigger_flag control whether mode 1 ISAC tracks are included. Each drone object includes position, sensor/gNB metadata, confidence, NR params, and optional aoa_deg.

Authorizations:
CognitoBearer

Returns time-ordered drone flight paths for historical replay (EGPD deployment). Queries DynamoDB for all points between startDateTime and endDateTime, grouped by tag and mode.
